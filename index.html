Prioritas Alokasi

Sistem mendahulukan store dengan rasio tinggi. Store Muku dan store dengan rasio terendah akan dikosongkan jika Qty Offline tidak mencukupi.

Cara Edit Manual

Anda bisa mengubah angka di kolom store secara manual. Kolom Sisa akan berubah merah jika alokasi manual Anda melebihi atau kurang dari Qty Offline.

Export Format

Gunakan tombol Export GSheet untuk mengunduh file CSV. Anda bisa langsung mengunggah file tersebut ke Google Drive.
